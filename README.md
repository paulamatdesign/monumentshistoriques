## Starter Commands

### Mac

- python3 -m venv .venv
- source .venv/bin/activate
- which pip
- pip install -r requirements.txt
- streamlit run Home.py
 -pip freeze > requirements.txt

### Windows

- python3 -m venv .venv
- ..venv\Scripts\activate.bat
- where pip
- pip install -r requirements.txt
- streamlit run Home.py
- pip freeze > requirements.txt
